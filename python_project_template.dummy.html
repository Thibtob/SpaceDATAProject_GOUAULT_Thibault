python_project_template.dummy module¶

Task management module.

This module provides functionality to add, list, mark as completed, and delete tasks, with storage in a JSON file.

python_project_template.dummy.add_task(tasks, task_name)¶

Add a new task.

Args:

tasks (list): The current list of tasks. task_name (str): The name of the task to add.

Effects:

Adds a new task to the list and saves it.

python_project_template.dummy.complete_task(tasks, task_index)¶

Mark a task as completed.

Args:

tasks (list): The current list of tasks. task_index (int): The index of the task to mark as completed (1-based).

Effects:

Updates the selected task’s status to completed and saves the list.

Raises:

IndexError: If the index is out of bounds.

python_project_template.dummy.list_tasks(tasks)¶

Display tasks.

Args:

tasks (list): The current list of tasks.

Effects:

Prints each task with its status (completed or not) to the console.

python_project_template.dummy.load_tasks()¶

Load tasks from the JSON file.

Returns:

list: A list of tasks, where each task is a dictionary with the keys ‘name’ (str) and ‘completed’ (bool).

python_project_template.dummy.main()¶

Main entry point of the program.

Displays an interactive menu to manage tasks, with options to add, list, mark as completed, or remove tasks.

python_project_template.dummy.remove_task(tasks, task_index)¶

Remove a task.

Args:

tasks (list): The current list of tasks. task_index (int): The index of the task to remove (1-based).

Effects:

Deletes the selected task from the list and saves it.

Raises:

IndexError: If the index is out of bounds.

tasks (list): A list of tasks to save.
